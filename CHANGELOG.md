# Changelog

## 1.0.0

- Initial public release of TeamPing
- Team-based ping system for Minecraft 1.21.1 NeoForge
- Radial ping wheel with custom icons
- Quick acknowledgement responses
- Server-authoritative networking
- Configurable ping duration, cooldown, and range

## Unreleased

- Smaller UI polish improvements
- Ongoing work on extra icons and radial customization
